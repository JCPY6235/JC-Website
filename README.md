## 🚀 Deployment on GitHub Pages

### Option 1: Using Your GitHub Account

1. **Create a GitHub Repository**
   ```bash
   git remote add origin https://github.com/YOUR-USERNAME/personal-website.git
   git branch -M main
   git push -u origin main
   ```

2. **Enable GitHub Pages**
   - Go to your repository settings
   - Scroll to "Pages" section
   - Select "main" branch as source
   - Your site will be live at: `https://YOUR-USERNAME.github.io/personal-website/`

### Option 2: Using `username.github.io` (Recommended)

For a custom domain without path:

1. Create a new repository named: `YOUR-USERNAME.github.io`
2. Push this code to that repository
3. Your site will be live at: `https://YOUR-USERNAME.github.io/`

Edit `css/styles.css` - Modify the CSS variables at the top:
```css

--secondary-color: #2c7cc7;
--accent-color: #f39c12;
```
